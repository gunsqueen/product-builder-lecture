# korea-admin-map-v2

새 프로젝트 재초기화 상태입니다.

## 현재 상태
- 기존 앱 코드는 `backup_old/`로 이동했습니다.
- 현재 루트의 앱은 새로 초기화된 최소 React + Vite + TypeScript 프로젝트입니다.
- 첫 화면에는 `korea-admin-map-v2 new project initialized` 문구가 표시됩니다.

## 환경변수
1. 루트에 `.env.local` 파일을 만듭니다.
2. `.env.example`을 참고해 값을 넣습니다.
3. `npm run dev` 전에 저장합니다.

## 실행
```bash
npm install
npm run dev
```

## 검증 포인트
- 브라우저 첫 화면에 `korea-admin-map-v2 new project initialized`
- `backup_old/` 폴더가 존재
- `src/` 아래가 새 최소 구조로 시작됨
