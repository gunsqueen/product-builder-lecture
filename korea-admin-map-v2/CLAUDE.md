# Project Context

이 폴더는 기존 시도 작업을 중단하고 새로 초기화한 `korea-admin-map-v2` 프로젝트다.

## 현재 목표
- 이전 앱과 완전히 분리된 새 시작점 확보
- 최소 실행 가능한 React + Vite + TypeScript 구조 제공
- 이후 전국 행정 데이터맵 기능을 새 구조 위에서 다시 구현

## 규칙
- `backup_old/`는 참고용 보관본이다.
- 현재 루트의 `src/`, `package.json`, `vite.config.ts`가 실제 새 앱 기준 파일이다.
- 첫 화면 문구는 반드시 기존 앱과 구분되게 유지한다.
